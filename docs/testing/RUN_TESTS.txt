====================================
üÜò TROUBLESHOOTING
====================================

Problem: "JAVA_HOME is not set"
Solution: Run the $env:JAVA_HOME command first

Problem: "mvnw.cmd not found"
Solution: Make sure you're in the niloy folder

Problem: "Build failed"
Solution: Make sure Docker is NOT required for tests
          (Unit and integration tests use H2, not PostgreSQL)
